Этот модуль позволяет динамически размещать декали в заранее отведенных местах, регионах (regions).
Задача этого модуля - разнообразить кабину путем добавления бумаг на стол бортинженера, которые можно сменять и редактировать прямо в процессе полета.

Это осуществляется путем отрисовки текстур, текста в определенные области в панели (panel.png). Этими областями являются:
Стол БИ, маленькая область слева - "eng_sm"
Стол БИ, остальная область стола - "eng_lg"
Область на приборной панели в середине, на месте заглушки - "dashboard"

Модуль загружает всю информацию из конфигурационных файлов в формате JSON в директории sets. Каждый файл определяет т.н. набор. Наборов может быть много, и их можно сменять
динамически в процессе полета, используя пункты меню: Tu-154M->Paperworks->Next set или Previous set. Также можно привязать эти команды к клавишам в меню настройки биндов сима. Они находятся в разделе tu154ce/paperworks. Пункт Update paperworks служит для перезагрузки конфигов после их редактирования.

Все ресурсы, такие как изображения или шрифта должны находиться в каталоге assets.

Базовый объект содержит два поля:
name - имя набора, отображается в меню.
regions - описание регионов

Каждый регион занимает определенную область в panel.png. Эта область определяется в поле "region":
x,y - координаты начальной точки
w - ширина области
h - высота

Учтите, что координаты в родной системе координат сима, в которой точка отсчета находится внизу слева:

        |---------- w -----------|
        +------------------------+--
        |                        | |
        |---------*              | h
        |         |              | |
        |         |              | |
    0   +------------------------+--

        0


Поле widgets содержит элементы, которые размещаются на регионах.
элементы могут быть трех типов:
image - изображение
text - статичный текст
custom - пользовательский лист бумаги любых размеров и с любым текстом

Отдельно от widgets стоит поле drefs. Здесь задаются датарефы, которые обновляются каждую секунду (настраивается).

Для удобства, в этой папке размещена текстура панели с определенными регионами и координатной сеткой. В будущем планируется GUI инструмент для редактирования наборов.
